Follow the 'Spiced Final project.pdf' to know about the app.

This app requires the next steps first::

1- Web Scraping (using Request, BeautifulSoup)
"scrap_moon_images.ipynb" save the Moon images in Jaxa_Kaguya_Images

2- Label on the images the craters (using labelImg)

3-Train the model to detect craters (using Yolov5)
"Moon_YOLO5.ipynb"

----------------------------
The App is in 'app' folder.


The app runs with 'app_script.py'

The files to test the app are in 'File_to_test'

app_video.mp4 & app_video2.mp4 show an overview of the app.
